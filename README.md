# AV2 - Sistema de Gestão de Compras

Avaliação 2 da disciplina de Frameworks - Luiz Henrique Mota Xavier

Aplicação Vue.js para gerenciar uma lista de compras em tempo real, utilizando reatividade, diretivas estruturais e manipulação de arrays.

## Como rodar

```bash
cd av2-luiz-mota-frameworks
npm install
npm run dev
```
